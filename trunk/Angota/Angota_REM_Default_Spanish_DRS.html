    SOP-0001 comenzarPedido(clienteID, comercialID, administradorID, datosPedido)
    Tipo del resultado El valor del identificador del objeto de la clase Pedido que se ha creado
    Versión 1.0 ( 17/05/2010 )
    Autores
    • Rafael Vergara Moreno
    Dependencias
  • [OBJ-0001] Control de compras
  • [OBJ-0002] Control de ventas
    • Descripción Operacion realizada cuando se comienza un pedido
    Parámetros DNICliente : Integer -- el dni correspondiente al cliente.
     DNIComercial : Integer -- el dni correspondiente al comercial que ha realizado el pedido.
     DNIAdministrador : Integer -- el dni correspondiente al administrador.
    numeroPedido : Integer -- el número que se le asigna al pedido.
    Expresiones de precondición Existencia de objeto de la clase Cliente en el sistema: Existe un objeto de la clase Cliente en el sistema tal que su identificador es DNICliente
     Existencia de objeto de Proveedor en el sistema: Existe un objeto de la clase Proveedor en el sistema tal que su identificador es proveedorID
    Expresiones de precondición (OCL) Existencia de objeto de la clase Cliente en el sistema: Pedido.allInstances() -> one(c|c.DNI = DNICliente)
    Existencia de objeto de Proveedor en el sistema: Pedido.allInstances() -> one(p|p.DNI = DNIProveedor)
    Expresiones de postcondición Creación de objeto de la clase Pedido: Se ha creado un objeto de la clase Pedido a la que se le han pasado como parámetros los datos del pedido, tal que su identificador es pedidoID
    Expresiones de postcondición (OCL) Creación de objeto de la clase Pedido: let ped : Pedido = Pedido.allInstances() -> any (numero = numPed), numPed: Integer = Pedido.allInstances() -> size()+1 in ped.oclIsNew() = true and ped.fecha = Date.Today() and return = ped.numero
    Comentarios Ninguno

    3.2 Registrar venta

    SOP-0012 añadirProducto(ventaID, productoID, cant)
    Versión 1.0 ( 11/06/2010 )
    Autores
    • Rafael Vergara Moreno
    Dependencias
  • [OBJ-0004] Control de almacén
  • [OBJ-0002] Control de ventas
    • Descripción Añade un producto a la venta
    Parámetros ventaID : Integer -- el identificador de una venta
     productoID : Integer -- el identificador de un producto
     cant : Integer -- la cantidad de un producto.
    Expresiones de precondición Exista la factura: Exista la factura con numero "ventaID"
     Exita el producto en el sistema: Exita el producto en el sistema y tenga cantidad suficiente
    Expresiones de precondición (OCL) Exista la factura: Factura.allInstances() -> one (f |f.numero = ventaID)
    Exita el producto en el sistema: Producto.allInstances() -> one (p| p.codigo = productoID and p.stock >= cant)
    Expresiones de postcondición Actualizar el producto y devolver información.: Actualizar el producto y devolver una tupla con el detalle del pedido y el subtotal, tambien quitar del stock la cantidad de producto agregada en la factura
    Expresiones de postcondición (OCL) Actualizar el producto y devolver información.: let fact : Factura = any ( numero = NumeroFactura), prod : Producto = any ( codigo = codigo), prodFact : ProductoFra = fact.productoFra.allInstances() -> (idProductoFra = IDPF), IDPF : Integer = ProductoFra = size()+1, prov : Proveedor = fact.Proveedor, in prodFact.oclIsNew() = true and prodFact.cantidad = cantidad and prodFact.precio = prod.precio and prod.stock = prod.stock@pre - cantidad and return = Tuple{descripcion:String = prod.descripcion,subTotal: Integer = prod.precio*cantidad}
